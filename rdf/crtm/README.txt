
This directory contains RDF files which do not contain any mapping
information in the source file. 

The mapping is provided by the file 'mapping.crtm' which is written 
in Compact RTM syntax.

Compact RTM is a non-RDF, non-standard syntax which can be used
to describe a RDF -> Topic Maps mapping compactly.

See <http://www.semagia.com/tr/crtm/>
